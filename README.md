## Denoak - Simple Deno Rest API built with oak

### Features

- [x] Logging(std/log)
- [x] Request Validation(computed_types)
- [x] CORS(cors)
- [x] Testing(superdeno)
- [x] Database Management(deno_mysql)
- [ ] Cache Layer
- [ ] Task Queue 

### Usage

```
$ deno run --allow-net --allow-env --allow-read main.ts
```

### Test

```
$ deno test --allow-all test/*
```
